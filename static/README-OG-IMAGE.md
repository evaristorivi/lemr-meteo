# Imagen Open Graph para Redes Sociales

## ¿Qué es?
La imagen Open Graph es la que aparece cuando compartes tu web en WhatsApp, Facebook, Twitter, LinkedIn, etc.

## Especificaciones requeridas
- **Nombre:** `og-image.jpg` (o `og-image.png`)
- **Tamaño:** 1200x630 píxeles (proporción 1.91:1)
- **Formato:** JPG recomendado (mejor compresión) o PNG
- **Peso máximo recomendado:** 300KB

## Contenido sugerido
La imagen debe incluir:
- Logo o icono del clima ☁️✈️
- Texto: "LEMR Meteo | La Morgal"
- Subtítulo: "Meteorología Aeropuerto Asturias"
- Fondo atractivo relacionado con aviación/meteorología
- Colores: usar la paleta del sitio (#66e3ff, #8d7cff, #0b1220)

## Herramientas para crear la imagen

### Opción 1: Canva (Fácil)
1. Ir a https://www.canva.com
2. Elegir "Custom Size" → 1200 x 630 px
3. Diseñar con textos y elementos
4. Descargar como PNG

### Opción 2: Figma (Profesional)
1. Crear frame de 1200x630
2. Diseñar
3. Exportar como PNG

### Opción 3: Photoshop/GIMP
1. Nuevo documento 1200x630 px
2. Diseñar
3. Guardar como PNG optimizado

## Dónde colocarla
Una vez creada, colócala en: `static/og-image.jpg` (o `static/og-image.png`)

## Verificar que funciona
Después de subir la imagen, verifica cómo se ve en:
- **WhatsApp:** Comparte el link en un chat
- **Facebook:** https://developers.facebook.com/tools/debug/
- **Twitter:** https://cards-dev.twitter.com/validator
- **LinkedIn:** Simplemente comparte el link
